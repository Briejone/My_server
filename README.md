# My Server

Простой HTTP-сервер на Go с использованием фреймворка Echo. Позволяет отправлять и получать сообщения через REST API.

## Возможности

- GET /messages — получение списка всех сообщений
- POST /messages — добавление нового сообщения

## Требования

- Go 1.18 или выше
